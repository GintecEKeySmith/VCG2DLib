The library provides basic functionalities with version 0.1v:

Draw lines, splines in various coloros
Draw Polygones (combination of lines and splines)
Fill Polygones (one color, color gradient (at the moment x has to be positive)

Coming in version 0.2
- Color gradients in neg. x directions
- Plotting of textures (filling polygons with texture maps)
- Color mixing (Transparency)

The library is based upon a paper:
http://www.academia.edu/36226207/A_novel_approach_to_2D_and_3D_computer_graphics._Reducing_pipe_intermingling
